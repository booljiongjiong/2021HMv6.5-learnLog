1. 标签选择器
    - 选出所有相同的标签
2. 类选择器
    - ```
        .类名 {
            样式属性:样式值;
            样式属性:样式值;
        }
      ```
    - 给标签添加多个类名 各个类名之间使用空格隔开 是一个标签拥有多个类名的样式 同时也方便统一修改
3. id选择器
    - 
    ```
        #id {
            样式属性:样式值;
            样式属性:样式值;
        }
    ```
4. 通配符选择器
    - 
    ```
    * {
         样式属性:样式值;
        样式属性:样式值;
    }
    ```
